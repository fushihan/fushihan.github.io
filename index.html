C++笔记

数组长度：数组的总占字节空间除以首元素所占字节空间。

N个元素和

使用指针可以使得函数返回多个值，返回最大和最小

使用指着进行字符串的复制、逆向输出等

获取个位数、十位数、百位数“

数组元素逆置

冒泡排序

总共排序次数=元素个数-1

比较次数=元素个数-当前轮数-1

new 返回的是开辟堆区的地址（指针）。

释放堆区的数组

函数的返回值不能作为函数重载的条件。

引用作为重载的条件

子类可以访问父类的保护成员，子类不可以访问父类的私有成员

  • struct 默认权限为公共
  • class 默认权限为私有

将成员属性设置为私有属性，用类内公共成员函数去访问私有成员。

在另一个类可以让另一个类作为本来类中的成员。

shift+tab 键 缩进

把类的声明放在头文件中，类的实现放在源文件中。

静态成员函数不能访问非静态成员变量。

静态成员变量，类内声明，类外初始化。

静态成员变量和静态成员函数有两种访问方式：①通过对象，②通过类名。

静态成员不占对象的内存空间。

每个空对象也应该有一个独一无二的内存空间，故内存大小为1.

成员变量和成员函数分开存储。

除了成员变量，其他都不占用对象内存

this 指针指向被调用的成员函数所属的对象。

this指针的用途：

  • 当形参和成员变量同名时，可用this指针来区分
  • 在类的非静态成员函数中返回对象本身，可使用return *this

this指针，可以用于链式编程。

堆区数据：程序员开辟，程序员清除。

类名（） 匿名对象，执行完这一行就被释放。

继承的好处：==可以减少重复的代码==

class A : public B;

A 类称为子类 或 派生类

B 类称为父类 或 基类

全部都被继承，只是私有成员被编译器隐藏，不可访问。

利用开发人员提示工具查看对象模型

  • 访问子类同名成员 直接访问即可
  • 访问父类同名成员 需要加作用域

子类和父类中有重名成员函数时，子类的的成员函数会屏蔽父类中重名的成员函数，必须加作用域访问。

多继承可能会引发父类中有同名成员出现，需要加作用域区分，故不太建议使用。

菱形继承的问题：

①多继承，重名二义性。加作用域解决。

②重复继承，数据重叠。利用虚继承。（在继承之前加virtual）

多态分为两类

  • 静态多态: 函数重载 和 运算符重载属于静态多态，复用函数名
  • 动态多态: 派生类和虚函数实现运行时多态

静态多态和动态多态区别：

  • 静态多态的函数地址早绑定 - 编译阶段确定函数地址
  • 动态多态的函数地址晚绑定 - 运行阶段确定函数地址

动态多态满足条件：

1.有继承关系

2.子类重写父类的虚函数

动态多态使用

父类的指针或者引用 执行子类对象

当类中有了纯虚函数，这个类也称为==抽象类==

抽象类特点：

  • 无法实例化对象
  • 子类必须重写抽象类中的纯虚函数，否则也属于抽象类

多态技术写代码，高内聚低耦合。

多态使用时，如果子类中有属性开辟到堆区，那么父类指针在释放时无法调用到子类的析构代码

解决方式：将父类中的析构函数改为虚析构或者纯虚析构

虚析构和纯虚析构共性：

  • 可以解决父类指针释放子类对象
  • 都需要有具体的函数实现

虚析构和纯虚析构区别：

  • 如果是纯虚析构，该类属于抽象类，无法实例化对象
